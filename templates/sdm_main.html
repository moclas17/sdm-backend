0xpo.app Backend Server

Este servidor verifica la autenticidad de tags NFC NTAG 424 DNA mediante validación criptográfica.

Endpoint Principal: /

Verifica que los parámetros recibidos de un tag NFC sean correctos validando:

  • UID: Identificador único del tag
  • CTR: Contador de lecturas del tag
  • CMAC: Firma criptográfica (CMAC) para validar autenticidad

Ejemplo de uso:

/?uid=041E3C8A2D6B80&ctr=000006&cmac=4B00064004B0B3D3

Si los parámetros son válidos, mostrará la información del tag. Si no, mostrará un error.
